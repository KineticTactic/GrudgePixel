> Quick Start

1. Getting Started

When you launch GrudgePixel, you'll be presented with three canvas size options:

  • 8 x 8
  • 16 x 16
  • 32 x 32

Click on your preferred size to begin.

2. Interface Overview

The main interface consists of:

  • Main canvas (420x420 pixels)
  • Color palette (right side)
  • Animation preview (top right)
  • Status bar (bottom)

3. Drawing Tools

3.1 Pencil

Left-click and drag on the canvas to draw. The pencil size can be adjusted.

3.2 Eraser

Right-click and drag on the canvas to erase.

3.3 Color Picker

Middle-click on a pixel to select its color.

4. Color Selection

Click on a color in the palette to select it. Use the 'P' key to switch between three different color palettes.

5. Canvas Navigation

Key Function
G Toggle grid lines
X Toggle center lines
M Toggle mirror mode
[ Decrease pencil size
] Increase pencil size
C Clear current frame

6. Frame Management

Key Combination Function
Ctrl + N Create a new frame
Left Arrow Go to previous frame
Right Arrow Go to next frame

7. Animation

Key Function
Space Play/Pause animation
Up Arrow Increase animation speed
Down Arrow Decrease animation speed

8. File Operations

8.1 Importing Images

Drag and drop an image file onto the GrudgePixel window to import it. The image will be automatically resized to fit the current canvas size.

8.2 Exporting

Press Ctrl + E to export all frames as individual PNG files. Files will be saved in a new directory named "session_[timestamp]".

9. Status Information

The status bar at the bottom of the screen provides the following information:

  • Current pixel coordinates
  • Current tool or action status
  • Mirror mode indicator (M)
  • Current pencil size
  • Current color preview
  • Animation status (Paused/Playing)
  • Current editing frame number
  • Animation speed (FPS)
  • Total number of frames

10. Tips and Tricks

  • Use mirror mode (M) for symmetrical designs.
  • Experiment with different color palettes using the 'P' key.
  • Create animations by using multiple frames and adjusting the playback speed.
  • Use the grid and center lines for precise pixel placement.
  • Adjust the pencil size for different effects, from fine details to larger areas.

11. Troubleshooting

If you encounter any issues:

  • Ensure you have the latest version of GrudgePixel.
  • Check that all required files (palettes, textures) are in the correct directories.
  • If the program crashes, try restarting it and reducing the number of frames or canvas size.
